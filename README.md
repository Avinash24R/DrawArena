# DrawArena     

Real-Time Multiplayer Drawing Game| React+ WebSockets + Docker-Based System          

- Designed and developed a real-time multiplayer drawing and guessing game 
- Built real-time drawing synchronization using WebSockets (Socket.IO) with event-based communication 
- Engineered backend room manager and game state engine for round handling, scoring, and player lifecycle 
- Containerized frontend and backend services using Docker and Nginx, orchestrated with Docker Compose 
- Integrated PostgreSQL and pgAdmin for data persistence and database management
